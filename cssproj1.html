CSS stands for Cascading Style Sheets.

It is used to style and design web pages.

CSS controls colors, fonts, spacing, and layouts.

It separates content (HTML) from presentation.

CSS makes websites more attractive and user-friendly.

It supports responsive design for different devices.

There are three types: inline, internal, and external CSS

External CSS allows reusing the same styles across pages.

CSS is an essential part of modern web development.
